# PathFinder
 **Цель:** создать интеллектуальную платформу для исследования городов, которая объединяет местных жителей и туристов через обмен знаниями о интересных местах и маршрутах.

## Техническое задание
### Бизнес-требования

**1. Для пользователей:**
 + Открытие новых мест в городе
 + Планирование пешеходных маршрутов
 + Обмен опытом через отзывы и рейтинги
 + Сохранение избранных мест и маршрутов
 
**2. Для модераторов:**
 + Контроль качества контента
 + Верификация пользовательских добавлений
 + Управление категориями и тегами
 + Функциональные требования
   

**3. Управление пользователями:**
 + Регистрация/аутентификация (JWT)
 + Ролевая модель (USER, MODERATOR, ADMIN)
 + Профиль пользователя с историей активности
   

**4. Система точек интереса (POI)**:
 + CRUD операции для POI
 + Геопоиск в радиусе
 + Категоризация и фильтрация
 + Система рейтингов и отзывов
 + Модерация пользовательского контента

 **5. Система маршрутов:**
 + Создание кастомных маршрутов
 + Расчет расстояния и времени
 + Публикация/приватность маршрутов
 + Система избранного

 **6. Административный функционал:**
 + Панель управления
 + Статистика платформы
 + Управление пользователями
 + Модерация контента
 ### Нефункциональные требования
 + Производительность: отклик API < 200мс
 + Поддержка 1000+ concurrent пользователей
 + Геопространственные запросы < 500мс
 + 99.9% uptime
---

## Стек технологий

**Backend:** `Java 21, Spring Boot 3.2+`  
**База данных:** `PostgreSQL 15+ with PostGIS`  
**ORM:** `Hibernate 6.4+ with Spatial types`  
**Безопасность:** `Spring Security 6, JWT`  
**Документация:** `OpenAPI 3 (Swagger)`  
**Тестирование:** `JUnit 5, Testcontainers, Mockito`  
**Сборка:** `Gradle 8+`  
**Контейнеризация:** `Docker, Docker Compose`  
**Мониторинг:** `Spring Boot Actuator, Micrometer`

## Архитектура веб-приложения
#### Frontend компоненты:
1. Главная страница с интерактивной картой
2. Панель поиска и фильтров
3. Детальные страницы POI
4. Конструктор маршрутов
5. Личный кабинет пользователя

 + UI/UX особенности:
 + Mobile-first дизайн
 + Progressive Web App _(PWA)_
 + Оффлайн-режим для карт
 + Интуитивный конструктор маршрутов
